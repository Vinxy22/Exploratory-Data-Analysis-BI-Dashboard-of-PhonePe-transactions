# Exploratory-Data-Analysis-BI-Dashboard-of-PhonePe-transactions
This project explores and visualizes PhonePe transaction data across 29 states, covering over 10M+ records. The goal was to uncover regional transaction patterns, user behavior, and growth trends using Python for data analysis and Power BI for interactive dashboards.
Key Highlights
- Conducted Exploratory Data Analysis (EDA) with Python (Pandas, NumPy, Matplotlib, Seaborn) to identify transaction spikes, seasonal trends, and anomalies.
- Built a Power BI dashboard with drill‑through KPIs, enabling stakeholders to filter by region, transaction type, and time period.
- Revealed regional insights such as higher transaction volumes in metro states compared to Tier‑2 regions.
- Automated data cleaning and preprocessing, improving dataset quality and reducing reporting time by 70%.
- Delivered actionable insights that can guide marketing strategies, regional growth planning, and customer engagement initiatives.
Tools & Skills Used
- Python: Pandas, NumPy, Matplotlib, Seaborn
- Power BI: DAX, drill‑through KPIs, interactive dashboards
- Data Analytics: EDA, trend analysis, KPI design, business storytelling
